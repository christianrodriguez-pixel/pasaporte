OBJETIVO
Reconstruir esta sección como una fila de insignias / badges de progreso clara, atractiva y coherente con el sistema de gamificación de Pasaporte Joven.

La sección debe comunicar:
- logros desbloqueados
- logros en progreso
- logros bloqueados
- colección / progreso personal

Debe verse premium, viva y limpia.
No debe verse administrativa.
No debe verse infantil.
No debe verse como stickers aleatorios.

ESTRUCTURA DE LA SECCIÓN
La sección debe incluir:

1. Header row
- título a la izquierda: “Insignias”
- acción a la derecha: “Ver todas”
- “Ver todas” con acento rojo/vino y chevron pequeño a la derecha

2. Fila horizontal de insignias
- una sola fila visual con 6 badges
- cada badge con:
  - icono / símbolo
  - nombre corto
  - estado o progreso debajo

INSIGNIAS QUE DEBE MOSTRAR
1. Primer check-in

   estado: Completada

   estado: 2/5

   estado: 0/3
6. Beneficiario estrella
   estado: 0/10

JERARQUÍA VISUAL OBLIGATORIA
Debe diferenciarse claramente entre tres estados:

A. DESBLOQUEADA / COMPLETADA
- más color
- más contraste
- más presencia
- sensación de premio real

B. EN PROGRESO
- visible y con intención
- no apagada
- debe insinuar avance
- puede mostrar progreso parcial mediante color parcial, anillo parcial o pequeño indicador visual

C. BLOQUEADA
- más tenue
- más desaturada
- más translúcida o monocroma
- todavía legible, pero claramente secundaria

ESTILO DE BADGES / INSIGNIAS
Quiero badges tipo emblema premium, no iconos sueltos.
No uses emojis.
No uses iconografía genérica administrativa.
No uses círculo plano simple como solución principal.

Cada insignia debe sentirse como una pieza coleccionable del sistema.

FAMILIA VISUAL DE INSIGNIAS
- misma familia formal para todas
- mismo nivel de detalle
- mismo tipo de borde
- misma lógica de luz y sombra
- coherencia total entre desbloqueadas, en progreso y bloqueadas

Forma sugerida:
- badge tipo hexágono, escudo suave o medallón facetado
- no hace falta que todas tengan exactamente la misma silueta si mantiene consistencia visual
- deben verse como “badges” y no solo como icon wrappers

TAMAÑO Y LAYOUT
- cada insignia debe ocupar el mismo ancho visual
- icono badge arriba
- nombre abajo
- estado/progreso en una tercera línea
- texto centrado
- espaciado consistente
- evitar que el nombre quede muy pegado al estado

DIMENSIONES SUGERIDAS
- badge visual: 54px a 72px aprox
- separación entre badge y título: generosa
- separación entre título y estado: menor pero clara
- toda la fila debe respirar

ICONOGRAFÍA POR INSIGNIA
Usa símbolos premium, limpios y claros:

1. Primer check-in
- símbolo: pin de ubicación / marcador
- estado: desbloqueada
- este badge debe sentirse como logro inicial importante
- puede llevar un chip pequeño “Nueva”

2. Racha de 7 días
- símbolo: flama
- estado: en progreso
- progreso: 5/7
- este badge debe usar más energía visual en rojo/vino
- debe verse activo, no bloqueado

3. Participante activo
- símbolo: rayo / energía / participación
- estado: desbloqueada
- debe sentirse como logro ganado

4. Explorador cultural
- símbolo: edificio cultural / museo / columna
- estado: en progreso
- progreso: 2/5
- no debe verse apagado del todo

5. Ruta completada
- símbolo: mapa / ruta / camino con pin
- estado: bloqueada o no iniciada
- progreso: 0/3
- más tenue

6. Beneficiario estrella
- símbolo: estrella
- estado: bloqueada o no iniciada
- progreso: 0/10
- más tenue

COLORES Y SEMÁNTICA
Usa esta lógica:

DESBLOQUEADAS / COMPLETADAS
- dorado / champagne / arena como color principal del badge
- detalles con luz cálida
- sensación premium y de logro
- estado “Completada” en color dorado o cálido

EN PROGRESO
- usar combinación de dorado y rojo/vino según el tipo
- “Racha de 7 días” puede ir más hacia rojo/vino
- “Explorador cultural” puede ir más neutra con una pista clara de progreso
- el progreso debe sentirse activo

BLOQUEADAS
- gris cálido / plata tenue / marfil apagado
- menos saturación
- menos contraste
- si quieres, candado pequeño discreto, pero no obligatorio
- no oscurecerlas tanto que se pierdan

TIPOGRAFÍA
- título de la sección: fuerte y claro
- nombres de insignias: pequeños pero legibles
- estado/progreso: todavía más pequeño
- “Completada” debe verse como estado positivo
- “5/7”, “2/5”, etc. deben leerse fácil

AJUSTES DE TEXTO

- Usa exactamente “Ver todas” como acción derecha
- No cambies los nombres de las insignias
- No cambies los estados

TRATAMIENTO DE PROGRESO
Para insignias en progreso, no quiero solo un número debajo.
Quiero que el badge sugiera progreso visualmente.

Puedes resolverlo con una o varias de estas opciones:
- anillo parcial
- borde parcialmente coloreado
- fill parcial
- highlight de progreso
- pequeña barra o microindicador integrado al badge

No hagas barras horizontales debajo del badge.
El progreso debe integrarse al badge mismo o a su contorno.

TRATAMIENTO DE “NUEVA”
Solo la primera insignia desbloqueada puede llevar un chip pequeño “Nueva”.
Debe ir arriba del badge o en una esquina del badge.
No hagas el chip gigante.
Debe verse como un acento.
