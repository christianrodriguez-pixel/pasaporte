Trabaja sobre la tarjeta actual de “Mi Pasaporte” dentro de la app “Pasaporte Joven” y genera una nueva iteración del sistema de credenciales por nivel.

No quiero rediseñar por completo la experiencia ni romper la estructura base actual.
Quiero evolucionar la tarjeta existente para que se vea más refinada, mejor jerarquizada y más premium, y además construir variantes por nivel: Black, Gold, Premium y Platinum.

Usa:
- la paleta de colores adjunta
- la tipografía Encode Sans
- los logos y assets adjuntos
- el mismo lenguaje general del producto

Objetivo:
Conservar la base visual y funcional de la tarjeta actual, pero mejorar:
- jerarquía de información
- claridad de lectura
- sofisticación visual
- diferenciación entre niveles
- integración sutil de una estética inspirada en Apple Liquid Glass

Importante:
No quiero glasmorfismo puro.
No quiero una tarjeta recargada.
No quiero una interfaz demasiado brillante o futurista.
Quiero solo destellos sutiles de una lógica tipo Apple Liquid Glass:
- capas con profundidad ligera
- reflejos muy discretos
- superficies pulidas
- separación elegante entre bloques
- sensación de materialidad premium
- translucidez muy controlada solo en detalles
- bordes suaves con highlights sutiles
- blur mínimo y solo donde aporte valor

La credencial debe seguir sintiéndose:
- institucional
- joven
- confiable
- premium
- clara
- moderna

Base del diseño:
Conserva la estructura general de la tarjeta actual:
- foto del usuario
- nombre
- ubicación
- bloque de puntos
- datos de nacimiento
- edad
- vigencia
- CURP
- institución
- folio o identificador inferior

Pero reorganiza y refina la jerarquía de información para que la lectura ideal sea:

1. Identidad del usuario
- foto
- nombre
- ubicación

2. Estatus y progreso
- nivel de tarjeta
- puntos acumulados
- progreso al siguiente nivel

3. Datos principales
- vigencia
- edad
- institución

4. Datos administrativos
- nacimiento
- CURP
- folio

Quiero que la tarjeta se sienta más ordenada visualmente y que no todos los bloques compitan por el mismo peso.

Mejoras de interfaz que quiero en la tarjeta base:

1. Bloque de puntos y progreso
Convierte este bloque en una mini hero section dentro de la tarjeta.
Debe incluir:
- puntos acumulados claramente jerarquizados
- nivel actual
- barra de progreso refinada
- texto de meta siguiente mejor redactado
Ejemplos:
“180 pts para Gold”
“Próximo nivel: Platinum”
“Te faltan 320 pts para Platinum”
- una superficie más rica y elegante
- un tratamiento visual ligeramente más premium que el resto de los bloques
- aquí es donde mejor puede vivir un toque sutil de materialidad tipo Liquid Glass

2. Bloques de datos
Refina los contenedores internos para que no se vean todos iguales.
Quiero:
- mejor jerarquía entre label y valor
- más aire
- mejor balance de tamaños
- menos sensación de ficha plana
- algunos datos con más prioridad que otros

3. CURP
Mantener visible, pero con tratamiento más administrativo y menos protagonista.
Puede verse un poco más técnico, limpio y ordenado.

4. Institución
Darle mejor tratamiento visual.
Debe sentirse importante y bien integrada, no como una simple línea de texto.

5. Patrón de fondo / ondas
Mantener el lenguaje visual de la tarjeta, pero hacerlo más refinado.
Quiero que el patrón se sienta más premium y menos decorativo.
Puede tener más profundidad y sutileza.

6. Bordes y capas
Aplicar detalles sutiles de materialidad:
- highlights suaves
- separación de capas ligera
- reflejos discretos
- superficies internas un poco más pulidas
Todo muy contenido.

Sistema de niveles:
No quiero un rediseño total por cada nivel.
Quiero un sistema consistente con un mismo layout base.
Las variantes deben diferenciarse por:
- color
- materialidad
- tratamiento del badge de nivel
- intensidad de detalle
- sofisticación visual
- pequeños cambios en acentos y acabados

Genera estas variantes:

1. Black
Sensación:
- exclusiva
- sobria
- elegante
- seria
- premium
Dirección visual:
- base negra o carbón profunda
- muy poco color saturado
- acentos sutiles en dorado tenue, champagne o vino oscuro
- textura muy discreta
- apariencia más mate y sólida
- no debe verse solo “oscura”, sino lujosa

2. Gold
Sensación:
- cálida
- aspiracional
- premium
- amigable
Dirección visual:
- dorado elegante, no amarillo chillante
- acentos arena y champagne
- brillo suave
- materialidad cálida
- badge de nivel más distinguible
- sensación aspiracional pero equilibrada

3. Premium
Sensación:
- energética
- destacada
- aspiracional
- más intensa
Dirección visual:
- usar la gama vino / rojo profundo / rojo acento de la paleta
- contraste sofisticado
- no debe verse estridente
- más presencia visual que Gold
- detalles más expresivos sin perder elegancia

4. Platinum
Sensación:
- máxima sofisticación
- refinada
- limpia
- técnica
- exclusiva
Dirección visual:
- plata cálida / titanio / grafito claro / metálico suave
- más luminosa que Black
- menos saturada que Premium
- más refinada que Gold
- esta variante puede ser la que más aproveche los destellos de materialidad tipo Liquid Glass
- usar highlights delicados, bordes pulidos, capas refinadas y micro reflejos sutiles
- debe sentirse la más premium de todas

Badge de nivel:
Cada variante debe tener un badge de nivel consistente con el sistema, pero con personalidad propia.
Quiero que el badge:
- se vea más premium
- esté mejor integrado
- refleje el tier sin recargar la tarjeta
- pueda ubicarse en la parte superior como sello o insignia

Quiero que me entregues:
1. una tarjeta base refinada
2. versión Black
3. versión Gold
4. versión Premium
5. versión Platinum

Además, muestra la evolución del sistema con coherencia:
- mismo layout base
- misma estructura
- diferencias visuales claras entre niveles
- no depender solo del color
- cada nivel debe sentirse como una mejora real del anterior

Lineamientos UX/UI:
- la tarjeta debe seguir siendo fácil de leer
- el nombre y la identidad del usuario deben ser muy claros
- el bloque de puntos debe tener un rol más protagónico
- la tarjeta debe sentirse como un objeto digital importante
- la lectura debe ser rápida, ordenada y confiable
- no sobrecargar de ornamentos
- no convertir el diseño en algo bancario genérico
- no volverlo una tarjeta de lujo vacía; debe seguir siendo útil y funcional

Resultado esperado:
Quiero un sistema de credenciales escalable, elegante y coherente, con refinamiento visual claro, mejor jerarquía de datos y variantes Black, Gold, Premium y Platinum que se distingan por materialidad, detalle y carácter, no solo por color.
